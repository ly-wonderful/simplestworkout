# SimplestWorkout

A clean, no-nonsense iOS workout tracker. Plan your week, log your lifts, get stronger.

## Features

- **Weekly Plans** — Organize your entire routine in one plan with labeled days (Push, Pull, Legs, etc.)
- **Auto Day Detection** — Tap Start and the app loads today's exercises automatically
- **Live Timer** — Tracks your session duration from start to finish
- **Previous Session Recall** — See your last reps and weight as gray placeholders in every field
- **Exercise Database** — Browse and search from thousands of exercises via wger.de
- **Full History** — Review past sessions with duration, exercises, sets, and weights
- **Sample PPL Program** — Import a ready-made Push/Pull/Legs program to get started instantly

## Tech Stack

- Swift & SwiftUI
- SwiftData for local persistence
- wger.de API for exercise database
- No third-party dependencies

## Requirements

- iOS 26.0+
- Xcode 26.0+

## Privacy

All data stays on your device. No accounts, no ads, no tracking. See [Privacy Policy](apps/SimplestWorkout/PRIVACY.md).
